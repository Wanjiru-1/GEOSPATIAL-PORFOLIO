Femicide Hotspot Mapping — Results
County-level analysis using Femicide Count Kenya dataset (2016–2024), cleaned and geocoded with Python + SQL
Key finding: Nairobi, Nakuru, and Kiambu account for 51% of all recorded femicide cases — urban density and economic pressure are significant correlating variables

Data cleaning removed duplicates and standardised county names using fuzzy matching. Correlation analysis with population density (KNBS 2019 census) and poverty index (Kenya MPI).
